Simple React app to manage your expenses.

Project goal: practice working with components, states, events, 
data flow between components, conditional content and rendering lists.

To make the app work:
1. Clone the repository
2. Install Node.js
3. Open the project (VS Code or your fav editor)
4. Open local terminal (in the repository location .../ReactExpensesControlApp)
5. Run "npm install" command and wait
6. Run "npm start" command and enjoy the app
